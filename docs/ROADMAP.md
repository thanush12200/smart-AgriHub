# Future Feature Roadmap

## Phase 1 (Current)
- Farmer/admin auth and role-based access
- Crop + fertilizer inference APIs
- Chatbot with confidence fallback
- Weather + alerts + dashboard analytics

## Phase 2 (Next)
- IoT sensor ingestion (soil moisture, pH, EC)
- Push notifications (SMS/WhatsApp)
- Multi-region crop price intelligence
- Better PDF/Excel reporting engine

## Phase 3 (Scale)
- MLOps: model drift detection + auto retraining
- Geo-spatial farm map layer and satellite insights
- Offline sync queue for low-connectivity mobile app
- Voice bot in multiple Indian languages (ASR + TTS)

## Phase 4 (Enterprise)
- Tenant-based startup accounts
- SLA monitoring and audit trails
- Policy-based access and compliance toolkit
